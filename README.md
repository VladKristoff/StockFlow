## Основной функционал
* **Учет ТМЦ:** Управление справочниками номенклатуры и контрагентов.
* **Документооборот:** Формирование, проведение и хранение приходных/расходных накладных.
* **Генерация:** Автоматическое создание печатных форм в формате .docx.
